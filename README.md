# 🎵 Music Playlist Web Application

A simple and responsive music playlist web application that allows users to browse and play songs through an interactive interface. This project demonstrates the fundamentals of front-end web development using HTML, CSS, and JavaScript.

---

## 📌 Features

- 🎶 Display a list of songs
- ▶️ Play and pause music
- ⏭️ Next and Previous song controls
- 📱 Responsive user interface
- 🎨 Clean and user-friendly design
- ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

• HTML5
• CSS3
• JavaScript
• PHP
• MySQL
